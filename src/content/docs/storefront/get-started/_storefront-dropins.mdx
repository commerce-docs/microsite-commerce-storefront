Dropins are full-featured shopping components. They are not primitive components like Carousels and Galleries. Dropins _define_ the storefront shopping experience. Examples include, Product Details, Carts, Checkout, Authentication, Accounts, and more. This section provides an introduction to dropin essentials.

## Vocabulary

<Vocabulary>

### Container

A container is a template for a a dropin.

### Slot

A slot is a placeholder for a components and functions.

### Component

A component is a reusable piece of code that can be used in multiple places.

### Function

A function is a reusable piece of code that can be used in multiple places.

### Configuration

Configuration is the settings that define how a dropin works.

### Customization

Customization is the process of changing the look and feel of a dropin.

### Integration

Integration is the process of adding a dropin to a website.

### Extension

An extension is a piece of code that adds functionality to a dropin.

### API

An API is a set of rules that allows different software applications to communicate with each other.

### SDK

An SDK is a set of tools and libraries that allow developers to create applications for a specific platform.
