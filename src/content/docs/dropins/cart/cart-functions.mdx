The Cart dropin provides API functions: `addProductsToCart`, `createEmptyCart`, `getCartData`, `getEstimatedTotals`, `getEstimateShipping`, `getStoreConfig`, `graphql`, `initializeCart`, `resetCart` and `updateProductsFromCart`. These functions allow developers to retrieve and display detailed cart information dynamically.

## addProductsToCart

This function adds products to a cart. It takes `sku`, `quantity`, and `optionsUIDs` as parameters.

```javascript
import { addProductsToCart } from '@/cart/api/addProductsToCart';

addProductsToCart(
  Array<{
    sku: string;
    quantity: number;
    optionsUIDs?: string[];
  }>
);
```

## createEmptyCart

A function that returns a new empty cart id.

```js
import { createEmptyCart } from '@/cart/api/createEmptyCart';

createEmptyCart();
```

## getCartData

A function that returns current cart data.

```js
import { getCartData } from '@/cart/api/getCartData';

getCartData();
```

## getEstimatedTotals

A function that returns estimated totals for cart based on an address. It takes an `address` parameter.

```js
import { getEstimatedTotals } from '@/cart/api/getEstimatedTotals';

getEstimatedTotals([
  {
    address: {
      countryCode: string;
      postcode?: string;
      region?: {
        region?: string;
        code?: string;
        id?: string;
      }
    }
  }
]);
```

## getEstimateShipping

A function that returns estimated shipping methods based on country id. It take `countryCode`, `postcode`, and `region.code` parameters.

```js
import { getEstimateShipping } from '@/cart/api/getEstimateShipping';

getEstimateShipping(
  {
    countryCode: string;
    postcode?: string;
    region?: {
      code?: string;
    }
  }
);
```

## getStoreConfig

A function that returns information about a store's configuration.

```js
import { getStoreConfig } from '@/cart/api/getStoreConfig';

getStoreConfig();
```

## initializeCart

A function that initialize either a Guest or Customer Cart.

```js
import { initializeCart } from '@/cart/api/initializeCart';

createEmptyCart();
```

## resetCart

A function that resets the cart.

```js
import { resetCart } from '@/cart/api/resetCart';

resetCart();
```

## updateProductsFromCart

A function that updates products in cart. It take `uid` and `quantity` parameters.

```js
import { updateProductsFromCart } from '@/cart/api/updateProductsFromCart';

updateProductsFromCart([
  {
    uid: string, quantity: number
  }
]);
```
