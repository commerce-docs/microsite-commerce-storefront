This section describes the containers and configuration options for the Checkout dropin.

## Big Picture

The Checkout dropin contains a single container that provides the layout and functionality for the checkout process. It is designed to be flexible and customizable, allowing you to create a checkout experience that meets your business needs.

## BillingForm configurations

The `Checkout` container provides the following configuration options:

    ['slots', 'string', 'object', 'No', 'An object containing slot overrides for customizing various parts of the component display.'],

  ]}

### cartId

The cart identifier reference to continue with the checkout process.

### routeHome

A callback function with an URI or URL to be redirected when the Go to Homepage links/buttons are clicked.

### routeCart

A callback function with an URI or URL to be redirected when the Go to Cart links/buttons are clicked.

### slots

Slots allow for the customization of component parts. Each slot accepts a "SlotProps" object with a context containing "data", "values", and a "valid" state. Custom actions and content can be added through these slots.

Visit the Slots page for slot details and usage information.

### preselectedCartData

The `preselectedCartData` property is an optional object that contains the initial cart information (address and shipping method fields) to preload the component. The object should follow the `PreselectedCartData` type:

```ts
export type PreselectedCartData = {
  address?: PreselectedAddressFields;
  shippingMethod?: PreselectedShippingMethod;
};

export type PreselectedAddressFields = {
  countryCode?: string;
  region?: string;
  postCode?: string;
};

export type PreselectedShippingMethod = {
  carrierCode: string;
  methodCode: string;
};
```

See the [example configuration](#example-configuration) for an example of how to use this property.

### Example configuration

The following example demonstrates how to configure the Checkout container:

```ts
return checkoutRenderer.render(Checkout, {
  cartId: cartRefFromSession,
  routeHome: () => {
    return '/';
  },
  routeCart: () => {
    return '/cart';
  },
  slots: {
    // See all Checkout slots in the next section
    Login: { },
    BillingForm: { },
    . . .
  },
  preselectedCartData: { 
    address: {
      countryCode: 'US',
      region: 'FL',
      postCode: '33130',
    },
    shippingMethod: {
      carrierCode: 'freeshipping',
      methodCode: 'freeshipping',
    },
  },
});
```
